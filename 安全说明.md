## 使用前请你做

1. 在 `.env` 填写**你自己的** Token、数据库、管理员 ID
2. 在 [@BotFather](https://t.me/BotFather) **撤销** GitHub 仓库里泄露的旧 Token
3. `PAYMENT_ADDRESS` 填**你自己的** TRON 收款地址
4. 勿将 `.env` 提交到 Git
