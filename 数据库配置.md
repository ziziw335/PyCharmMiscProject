# 数据库配置

支持两种模式：

| 模式 | 说明 |
|------|------|
| **SQLite**（默认） | `.env` 里 `DATABASE_URL=sqlite:///data/bot.db`，本机零配置 |
| **云 PostgreSQL** | Neon 等免费云库，多机/部署时用 |

## 连接云 PostgreSQL（Neon，约 3 分钟）

1. 打开 **https://neon.tech** → 注册 → **New Project**
2. 左侧 **Connection string** → 选 **Pooled connection** → 复制（`postgresql://...`）
3. 在 PyCharm 终端执行（把整段连接串贴在引号里）：

```bash
cd /Users/xiaoyang/PyCharmMiscProject
.venv/bin/python setup_cloud_postgres.py "postgresql://用户:密码@ep-xxx.neon.tech/neondb?sslmode=require"
```

4. 看到 `✅ 云 PostgreSQL 连接成功` 后，重启 **运行 Bot** / **运行 Web后台**

可选：把本地 `data/bot.db` 里的数据迁到云库：

```bash
.venv/bin/python setup_cloud_postgres.py "postgresql://..." --migrate-sqlite
```

## 自检

```bash
.venv/bin/python test_db.py
```

云库应显示 `✅ PostgreSQL 连接成功`。

## 常见错误

| 报错 | 处理 |
|------|------|
| DATABASE_URL is missing | `.env` 里没填连接串 |
| SSL 相关错误 | 确保 URL 末尾有 `?sslmode=require`，且 `PGSSLMODE=require` |
| connection refused | 连接串复制不完整，重新从 Neon 复制 |
