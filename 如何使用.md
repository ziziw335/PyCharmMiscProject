## 一、快速开始

### 1. 本地开发（PyCharm）

1. **File → Open** → 选择项目根目录（不要只打开 `.venv`）
2. 解释器选 **`.venv/bin/python`**
3. 复制并编辑 **`.env`**（参考 `.env.example`），至少填写：
   - `BOT_TOKEN` — [@BotFather](https://t.me/BotFather)
   - `BOT_OWNER_ID` — 你的 Telegram 数字 ID（[@userinfobot](https://t.me/userinfobot)）
   - `WEB_TOKEN` — 网页后台登录密码（请用长随机串，勿用弱密码）
4. 右上角运行：
   - **运行 Bot** → `app.py`
   - **运行 Web后台** → `web.py`（浏览器 `http://127.0.0.1:8081`，密码 = `WEB_TOKEN`）

### 2. 云服务器（推荐 7×24）

```bash
cd /Users/xiaoyang/PyCharmMiscProject
./deploy_to_vps.sh
```

或见下文「部署与维护」。  
**注意：** VPS 与 Mac 不要同时跑同一个 `BOT_TOKEN`，否则会抢消息。

### 3. 本机 + 公网账单页（ngrok，仅测试）

见 **`一键启动说明.txt`**：先 Web → 再 Bot → `ngrok http 8081` → 配置 `PUBLIC_WEB_BASE_URL`。

## 三、群内记账功能

### 3.1 开启 / 关闭

| 指令 | 说明 |
|------|------|
| `开始` / `开始记账` / `🔥 开始记账` | 开启本群记账 |
| `关闭记账` / `停止记账` | 关闭本群记账 |
| `上课` | 开启群发言（若已配置） |
| `下课` | 限制群发言（若已配置） |

### 3.2 记账指令（需为操作员或管理员）

| 类型 | 示例 |
|------|------|
| 入款 | `+1000`、`-1000`、`+7777u` |
| 带汇率 | `+1000/7.8` |
| 下发 | `下发5000`、`下发1000R` |
| 寄存 | `P+2000`、`P-1000` |
| 带备注 | `+1000 备注文字` |
| 指定对象 | `张三+1000`，或回复某人消息后发 `+1000` |
| 查账 | `账单`、`总账单`、`今日总账单`、`上个月总账单`、`/我` |
| 撤销 | `撤销`（撤销上一笔） |
| 清空今日 | 按 Bot 内「删除账单」说明的指令 |
| 快捷 | `+0` / `-0` / `0` → 显示今日账单 |

### 3.3 参数设置

| 指令 | 说明 |
|------|------|
| `设置汇率7.2` | 设置默认汇率 |
| `设置费率10` | 设置默认费率（%） |

### 3.4 本群操作员

| 操作 | 方法 |
|------|------|
| 添加操作人 | **回复**对方消息，发送 `添加操作人` |
| 移除操作人 | **回复**对方消息，发送 `删除操作人` / `移除操作员` |

### 3.5 群内按钮

- **查看本群账单** — 在群内直接发送今日账单（不强制打开网页）
- Bot 入群会自动登记群组；有消息后记入数据库

### 6.3 银行卡 / 四要素配置

详见 **`银行卡查询配置说明.md`**。

- **仅卡号** → 用「查卡」，可查 **开户行**，不能合法反查姓名手机。
- **四要素** → 用「四要素核验」，需在 `.env` 配置易源 / 云市场 APPCODE。

```env

HIBP_API_KEY=你的Key
```

## 七、Web 管理后台

### 7.1 登录

- 地址：`http://服务器IP:8081` 或配置的 `PUBLIC_WEB_BASE_URL`
- 密码：`.env` 中的 **`WEB_TOKEN`**（与 Telegram 内嵌链接的只读 token 不同）

### 7.2 主要页面

| 页面 | 功能 |
|------|------|
| 仪表盘 | 用户 / 订单统计 |
| 群组列表 | 群账单、**群成员**用户名、聊天记录 |
| 群组同步 | 从 Telegram 同步群信息 |

| 订单 | 审核通过 / 拒绝 |
| 私聊记录 | 用户与 Bot 私聊日志 |

### 7.3 交易历史链接安全

Bot 里「📜 交易历史」打开的链接使用 **群专属只读令牌**（`gv1.xxx`），**不是** `WEB_TOKEN`，他人拿到链接也只能看该群账单，进不了完整后台。

## 九、部署与维护（VPS）

### 9.1 首次部署

```bash
# Mac 上传
cd /Users/xiaoyang/PyCharmMiscProject
./deploy_to_vps.sh

```

```bash

```

## 十一、常见问题

**Q：关 Mac 后 Bot 还运行吗？**  
A：若已部署在 VPS 且 systemd 已 enable，**会继续运行**；不要在 Mac 上再开同一个 Bot。

**Q：交易历史里群不见了？**  
A：群列表来自数据库；换库后需迁移数据或在群里发消息 / 后台「群组同步」。

**Q：查用户名失败？**  
A：在服务器执行 `pip install sherlock-project` 后重启 Bot。

**Q：Web 后台进不去？**  
A：检查防火墙是否放行 **8081**，`WEB_TOKEN` 是否正确。

**Q：能否只凭卡号查姓名和手机？**  
A：**不能**（无合法公开接口）。请用「四要素核验」或合规付费 API，见 `银行卡查询配置说明.md`。
